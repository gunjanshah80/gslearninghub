Python Beginner to Builder Program

A 16-week hands-on Python learning journey designed to take students from basic programming concepts to building practical mini projects.

Program Details

📅 Duration: 16 Weeks

🗓️ Schedule: 3 classes per week, 1 hour per class

👦 Eligibility: Class 6 and Above

💻 Mode: Online / Offline Hands-on Coding Practice

This program is designed for students who want to build a strong foundation in Python programming
through step-by-step learning, hands-on coding exercises, practice challenges and a mini project using real-world data.

Prerequisites

  • Basic computer handling skills
  • Laptop is recommended for hands-on practice
  • No prior Python knowledge required

Course Overview

Foundation First

Students begin with programming vocabulary, Python setup, Jupyter Notebook, variables, data types and operators.

More Complex Constructs

Students move beyond basic syntax and learn conditions, loops, strings, lists and structured problem-solving through hands-on coding exercises.

Code Organization

Students learn functions, modules, libraries, file handling and error handling to write better programs.

Mini Project

Students apply their learning by building small real-world programs and presenting their work.

16-Week Course Schedule

Week 1 – Introduction to Python, Setup and Python IDEs

  • Introduction to Python and its real-world use
  • Installing Python on local machine
  • Understanding Python PATH and pip setup
  • Understanding Python installation and project folders
  • Introduction to popular Python IDEs and editors
  • Setup and First look at Jupyter Notebook
  • Running the first Python program

Week 2 – Introduction to Notebook, Setup and Practice

  • Installing and launching Jupyter Notebook
  • Understanding Jupyter Notebook interface
  • Working with Jupyter Notebook
  • Code cells and Markdown cells
  • Working with code cells and Markdown cells
  • Running cells using menu options and keyboard shortcuts
  • Creating the first structured Python notebook
  • Saving, renaming and organizing notebooks
  • Understanding the difference between .py files and .ipynb notebooks
  • Documenting Code using Markdown

Week 3 – Variables and Data Types

  • Understanding variables and constants
  • Working with integer, float, string and boolean values
  • Checking data type using type()
  • Making Python programs interactive using input()
  • Understanding that input() returns string data
  • Type conversion
  • Mini challenges and practice exercises

Week 4 – Operators and Expressions

  • Arithmetic operators
  • Relational operators
  • Logical operators
  • Expressions in Python
  • Strike rate calculator mini challenge
  • Mini challenges and practice exercises

Week 5 – Working with Strings

  • Creating and printing strings
  • String indexing and slicing
  • String methods
  • String formatting
  • Name and message formatting exercise
  • Mini challenges and practice exercises

Week 6 – Decision Making

  • Understanding conditions
  • if, elif and else
  • Nested conditions
  • Pass/fail and grading logic
  • Student result program
  • Mini challenges and practice exercises

Week 7 – Loops in Python

  • Why loops are used
  • for loop
  • while loop
  • Using range()
  • Number pattern exercises
  • Mini challenges and practice exercises

Week 8 – Lists and Tuples

  • Creating lists and tuples
  • Accessing elements
  • Adding and removing list items
  • Looping through lists
  • Favourite players list exercise
  • Mini challenges and practice exercises

Week 9 – Dictionaries and Sets

  • Key-value pairs
  • Creating dictionaries
  • Accessing and updating values
  • Introduction to sets
  • Student marks dictionary exercise
  • Mini challenges and practice exercises

Week 10 – Functions

  • Why functions are used
  • Creating functions using def
  • Parameters and return values
  • Reusable code
  • Calculator using functions
  • Mini challenges and practice exercises

Week 11 – Python Libraries

  • What are libraries?
  • Standard libraries
  • Using math, random and datetime
  • Installing external libraries using pip

Week 12 – File Handling

  • Reading files

Week 13 – Error Handling and Debugging

Week 14 – Working with JSON Data

Week 15 – Mini Project Development

  • Selecting a mini project idea
  • Planning inputs and outputs

Week 16 – Project Presentation and Recap

  • Final project completion
  • Code review and improvements
  • Student project presentation
  • Course recap
  • Next learning path after Python basics
  • Mini challenges and practice exercises

Mini Projects Students May Build

Student Report Card

Take subject marks, calculate total, percentage and grade using Python logic.

Number Guessing Game

Build a simple game using random numbers, loops and conditions.

Personal Expense Tracker

Record daily expenses and calculate total spending using lists and files.

Student Records Manager

Store and manage student details using dictionaries, files and JSON data.
