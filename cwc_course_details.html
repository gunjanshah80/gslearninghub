Course Details

Program: Coding Without Computers

Duration: 12 Weeks

Age Group: 8–14 Years

Mode: Off-Screen Coding with specially designed Flash Cards, Problem-solving Activities and various coding challenges.


12-Week Course Structure:

  • Understanding Program Flow using Flash Cards, Sequencing and breaking down tasks into smaller steps.

  • IPO Model (Input → Process → Output) using real-life examples.

  • Understanding Flash Card Vocabulary such as START, STOP, NEXT, INPUT, OUTPUT and PROCESS. Learning how to logically combine flash cards to create basic coding patterns.

  • Working with Variables and Constants. Creating variables, storing values and using them within programs.

  • Understanding Operators. Expanding vocabulary through story-based coding activities & mini-challenges with Arithmetic Operators (+, -, ×, ÷, %).

  • Expanding vocabulary through story-based coding activities & mini-challenges with Relational Operators (>, <, >=, <=, =, !=).

  • Begin practicing decision‑making with IF…THEN…ELSE constructs by working through multiple coding examples.

  • Explore Logical Operators AND, OR and NOT through fun activities and truth tables.

  • Understanding Loops and Repetition using LOOP, REPEAT and WHILE constructs.

  • Combining Conditions and Loops to solve more complex problems.

Hands-on Challenges & Activities:

Students will also work upon various coding challenges and problem-solving activities to build computational thinking and logical reasoning skills.

  • 🔍 Debugging Detective Challenge
    Find and fix mistakes in a given flash-card program.

  • ⚡ Fastest Program Builder Challenge
    Build a correct solution using flash cards in the shortest time.

  • 🧩 Program Sequence Builder Challenge
    Arrange shuffled program steps in the correct logical order.

  • 🤝 Pair Programming Challenge
    Work with a partner to design and test a solution together.

  • 🧠 Logic Detective Challenge
    Find the correct answer to a logic problem.
