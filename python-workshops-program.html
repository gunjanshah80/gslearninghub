Python Basics Workshop

A beginner-friendly hands-on workshop designed for students who are new to Python or have limited programming experience.

👦 Eligibility: Class 6 and Above

  • Why Python is in high demand
  • Python setup and installation on local machine
  • Python in fast-growing fields like AI, Data Analytics and Automation
  • Popular Python IDEs and coding environments
  • Working with Jupyter Notebook
  • Python variables and data types
  • Python operators and expressions
  • Working with strings
  • Decision-making using conditions
  • Repeating tasks using loops
  • Introduction to Python libraries
  • Topic-wise coding exercises and practice challenges

Python Advanced Workshop

A practical workshop for students who already know Python basics and want to strengthen their problem-solving skills and learn more powerful Python concepts used in real-world programming.

👨‍💻 Eligibility: Class 9 and Above

  • Quick revision of Python fundamentals
  • Advanced string manipulation and text processing
  • Working with regular expressions for pattern matching
  • Data Structures: Lists, tuples, dictionaries and sets
  • List comprehension and efficient coding techniques
  • Functions and reusable code
  • Lambda functions for short and simple operations
  • File handling in Python
  • Error handling and debugging basics
  • Working with JSON data in Python
  • Using Python standard libraries
  • Building a mini project using real-world data
  • Advanced coding challenges and practice problems
